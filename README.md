# Tic-tac-toe made with React using Typescript template.

## Status: Finished ✔️

## Available Scripts & instaling dependencies

### Using npm
```bash
    npm install 
    npm run start
```
### Using yarn
```bash
    yarn install
    yarn start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the your default browser.
